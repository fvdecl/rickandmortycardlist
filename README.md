# RickAndMortyCardList

**RickAndMortyCardList** — это Flutter-проект, который отображает список персонажей из [Rick and Morty API](https://rickandmortyapi.com/api/character) и позволяет добавлять персонажей в избранное.  
Приложение имеет две страницы:  

1. **Список персонажей** — отображает всех персонажей с их изображениями и именами.  
2. **Избранное** — отображает только те персонажи, которых пользователь отметил как избранные.  

Избранные карточки сохраняются в **IndexedDB** через пакет **idb_shim** (для Flutter Web).  

---

## Функциональность

- Просмотр персонажей с изображением и именем.  
- Добавление/удаление персонажей в избранное.  
- Хранение избранных карточек в IndexedDB.  
- Поддержка светлой и тёмной темы.  
- Сортировка избранных по имени.
